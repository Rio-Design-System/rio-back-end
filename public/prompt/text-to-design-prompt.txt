## 13. **ICONS - VECTOR NODES WITH PATH DATA**

Icons MUST be created using VECTOR nodes with `vectorPaths`. This is the proper way to create scalable icons in Figma.

### ⚠️ CRITICAL: SVG Path Syntax Rules

**IMPORTANT: Path data MUST have spaces between commands and numbers!**

✅ CORRECT: `"M 5 12 L 19 12"` (spaces after M, L and between numbers)
❌ WRONG: `"M5 12 L19 12"` (missing space after M and L)
❌ WRONG: `"M5,12 L19,12"` (avoid commas, use spaces)

**Path Command Format:**
- Every command letter (M, L, Q, C, Z Only) MUST be followed by a space
- Numbers MUST be separated by spaces
- Use spaces, NOT commas, between coordinates
- Dont use this letter (V, H, A)

### Vector Path Structure:
```json
{
  "name": "Icon Name",
  "type": "VECTOR",
  "x": 0,
  "y": 0,
  "width": 24,
  "height": 24,
  "fills": [],
  "strokes": [
    {
      "type": "SOLID",
      "visible": true,
      "opacity": 1,
      "blendMode": "NORMAL",
      "color": { "r": 0.2, "g": 0.2, "b": 0.2 }
    }
  ],
  "strokeWeight": 2,
  "strokeCap": "ROUND",
  "strokeJoin": "ROUND",
  "vectorPaths": [
    {
      "windingRule": "NONZERO",
      "data": "M 5 12 L 19 12"
    }
  ]
}
```

### SVG Path Commands Reference (use only this):
- **M x y** - Move to point (start new path) - Example: `M 5 12`
- **L x y** - Line to point - Example: `L 19 12`
- **C x1 y1 x2 y2 x y** - Cubic bezier curve - Example: `C 10 5 15 5 20 12`
- **Q x1 y1 x y** - Quadratic bezier curve - Example: `Q 12 5 19 12`
- **Z** - Close path (connect back to start)

### Icon Usage Tips:
1. **Standard size**: Use 24x24 for most icons, 16x16 for small, 32x32 or 48x48 for large
2. **Stroke weight**: Use 1.5-2 for 24px icons, adjust proportionally for other sizes
3. **Color**: Use stroke color for outline icons, fills for solid icons
4. **strokeCap and strokeJoin**: Use "ROUND" for friendlier icons, "SQUARE"/"MITER" for sharper look
5. **Scaling**: To scale icons, multiply all coordinates and dimensions by the scale factor
6. **Filled icons**: Use fills instead of strokes and adjust vectorPaths accordingly
