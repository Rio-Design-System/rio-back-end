ICONS — USE ICONIFY API (NOT VECTOR PATHS)

IMPORTANT — AVAILABLE ICONS FROM REFERENCE:
If an "AVAILABLE ICONS FROM REFERENCE" list is provided in the user message:
- Use those exact names for the icon nodes in your output (e.g., name the node "google" not "Google Icon")
- Still use RECTANGLE with IMAGE fill from Iconify as usual — the system will automatically restore the original icons after generation
- DO NOT call searchIcons for these — pick the closest matching Iconify icon visually and name the node with the exact reference name

When you need icons NOT in the reference list (or when no list is provided):

⚡ BATCH ALL ICON SEARCHES IN ONE ROUND:
Before generating the design, scan the entire layout and identify every icon you will need.
Then call searchIcons for ALL of them simultaneously in a SINGLE response — do NOT search incrementally across multiple rounds.

Each searchIcons call returns an array of icon ID strings (e.g. "mdi:home", "lucide:arrow-right").
Construct the icon URL from the ID using this pattern:
https://api.iconify.design/{prefix}/{name}.svg
where prefix and name are the two parts split by ':'.
Example: "mdi:home" → https://api.iconify.design/mdi/home.svg

Example result from searchIcons:
  { "icons": ["mdi:home", "mdi:home-outline", "mdi:home-variant"] }

Then use RECTANGLE with IMAGE fill:

```json
{
  "name": "Home Icon",
  "type": "RECTANGLE",
  "x": 0,
  "y": 0,
  "width": 24,
  "height": 24,
  "fills": [
    {
      "type": "IMAGE",
      "scaleMode": "FILL",
      "imageUrl": "https://api.iconify.design/mdi/home.svg"
    }
  ]
}
```

DO NOT use VECTOR nodes with vectorPaths for icons!
ALWAYS use RECTANGLE with IMAGE fill from Iconify URL!

Common Icon Prefixes:
- mdi: — Material Design Icons (mdi:arrow-right, mdi:check) ⭐ PREFERRED — use mdi: by default for most icons
- solar: — Modern, clean icons (solar:home-bold, solar:user-linear)
- lucide: — Lucide icons (lucide:settings, lucide:menu)
- heroicons: — Heroicons (heroicons:home, heroicons:user)
- tabler: — Tabler icons (tabler:home, tabler:search)

Icon Sizes:
- Small: 16x16
- Default: 24x24
- Medium: 32x32
- Large: 48x48

Coloring Icons:
To change icon color, append a `color` query parameter to the Iconify URL:
- White icon: https://api.iconify.design/mdi/home.svg?color=white
- Custom hex: https://api.iconify.design/mdi/home.svg?color=%231766F9
