When you need an icon in the design:

1. **FIRST**: Call the `searchIcons` tool with a keyword (e.g., "home", "arrow", "user")
2. **THEN**: Call the `getIconUrl` tool with the best matching icon from results
3. **FINALLY**: Use RECTANGLE with IMAGE fill:

```json
{
  "name": "Home Icon",
  "type": "RECTANGLE",
  "x": 0,
  "y": 0,
  "width": 24,
  "height": 24,
  "fills": [
    {
      "type": "IMAGE",
      "scaleMode": "FILL",
      "imageUrl": "https://api.iconify.design/solar/home-bold.svg"
    }
  ]
}
```

### Icon Sizes:
- Small: 16x16
- Default: 24x24
- Medium: 32x32
- Large: 48x48

### Coloring Icons:
To change icon color, append a `color` query parameter to the Iconify URL:
- White icon: `https://api.iconify.design/solar/home-bold.svg?color=white`
- Custom hex: `https://api.iconify.design/solar/home-bold.svg?color=%231766F9`
